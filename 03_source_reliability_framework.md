# 03. Source Reliability Framework

## Purpose

Risk work depends on the quality of sources. This framework helps distinguish between strong evidence, useful context, weak signals, and material that should not be used for decision-making.

## Source categories

| Category | Examples | Typical use |
|---|---|---|
| Official government travel advice | Foreign ministry travel pages, consular alerts | Baseline country-level risk and practical advice. |
| EU / international institutional sources | EU, UN, WHO, ECDC, Interpol/Europol public pages where relevant | Institutional context and public safety information. |
| Local official sources | Police notices, municipal alerts, airport/transport authority updates | Practical and localised disruptions. |
| Reputable media | Established outlets with named authors and editorial standards | Context and incident awareness. |
| Specialist security sources | OSAC, insurance/security advisories, assistance providers | Security context, where accessible and corroborated. |
| Public social media signals | Official accounts, verified organisations, public event channels | Early warning and situational awareness only. |

## Reliability scoring

| Score | Label | Use |
|---|---|---|
| A | Strong | Can support a briefing finding if relevant and current. |
| B | Good | Useful, normally corroborate if the point is important. |
| C | Contextual | Use carefully; do not rely on it alone for decisions. |
| D | Weak | Do not use for key findings unless verified elsewhere. |

## Verification checks

For each source, ask:

1. Who published it?
2. When was it published or updated?
3. Is it primary or secondary?
4. Does it directly support the claim being made?
5. Is the claim corroborated elsewhere?
6. Is the source geographically specific enough?
7. Is the source relevant to the travel purpose?
8. Does the source introduce bias, uncertainty, or missing context?

## Example source log entry

| Field | Example |
|---|---|
| Source name | Foreign Ministry Travel Advice - Lydora |
| Source type | Official government travel advice |
| Date checked | 2026-06-15 |
| Topic | Security and transport |
| Key finding | Notes increased petty crime near central transport hubs. |
| Reliability | A |
| Limitation | Country-level; limited neighbourhood detail. |
| Follow-up | Check local transport authority and recent incident reporting. |

## Analyst note

A strong source can still be incomplete. A weak source can still flag something worth checking. The analyst should avoid both overconfidence and excessive scepticism.
