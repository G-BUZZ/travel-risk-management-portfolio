# 01. Project Overview

## Project title

**Travel Risk Management OSINT Portfolio Project**

## Objective

The objective is to demonstrate a structured junior-level approach to supporting travel risk management tasks, including:

1. preparing a concise travel security briefing;
2. mapping accommodation options against practical safety criteria;
3. logging and rating public sources;
4. monitoring public signals without collecting unnecessary personal data;
5. analysing traveller feedback;
6. triaging urgent communications;
7. documenting limitations and escalation triggers.

## Intended audience

This project is designed for reviewers assessing entry-level capability in:

- security risk support;
- corporate security assistance;
- travel security coordination;
- open-source research;
- EU/international organisation internships;
- compliance-adjacent risk documentation.

## Scenario used

The project uses a **fictional non-EU destination: Port Azure, Republic of Lydora**.

The scenario is deliberately synthetic. It is realistic enough to show methodology, but it avoids making unsupported claims about a real country, city, hotel, group, or individual.

## What the project does not do

This project does not:

- identify real individuals;
- scrape social media platforms;
- bypass platform terms or privacy settings;
- make real threat assessments;
- provide operational security instructions;
- replace official travel advice;
- claim law enforcement or intelligence experience.

## Core outputs

- A sample travel risk brief.
- An accommodation mapping matrix.
- A source reliability framework.
- A traveller feedback analysis workflow.
- An urgent communication triage model.
- Python-generated scoring and summary outputs.


## Candidate positioning

This project is best presented as evidence of junior travel risk management support skills: structured documentation, source validation, accommodation mapping, security briefing support, traveller feedback analysis, and escalation logic. It should not be presented as operational security, intelligence work, or real-time threat assessment.
